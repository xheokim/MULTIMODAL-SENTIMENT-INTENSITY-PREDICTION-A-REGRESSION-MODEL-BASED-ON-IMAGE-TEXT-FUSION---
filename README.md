# MULTIMODAL-SENTIMENT-INTENSITY-PREDICTION-A-REGRESSION-MODEL-BASED-ON-IMAGE-TEXT-FUSION---
This project implements a multimodal regression pipeline to predict social media sentiment intensity on a 0–5 scale using the MVSA-Multiple dataset. The architecture utilizes a late-fusion approach, concatenating 768-D text features from BERT and 512-D visual features from ResNet18 into a lightweight MLP regressor.
Results show the multimodal model significantly outperforms unimodal baselines, achieving an MAE of 0.7141 and a Pearson correlation of 0.5518. Notably, fusion reduces MAE by 21.7% and improves correlation by 63.0% compared to the image-only model, proving that integrating visual and textual signals is crucial for capturing nuanced social media sentiment. This repository provides a reproducible end-to-end baseline for multimodal sentiment analysis.
